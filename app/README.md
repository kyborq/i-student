# iStudent

## Development

### 1. Clone repository

```
git clone https://github.com/kyborq/iStudent.git
cd iStudent
```

### 2. Install dependencies

```
npm install
```

### 3. Run

| Don't forget to connect your device to PC or run android emulator 

Start metro server

```
npm start
```

Run app

```
npm react-native run-android
```
